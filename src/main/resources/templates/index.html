Проект сервиса планирования задач - job4j_todo

Приложение для управления задачами (ToDo), которое позволяет:

    Редактировать описание, статус;

    Удалять завершенные или ненужные задачи;

    Просматривать список всех задач, с возможностью фильтрации по статусу (выполненные/невыполненные);

Это простое, но функциональное приложение помогает эффективно организовывать повседневные дела и отслеживать их выполнение.
